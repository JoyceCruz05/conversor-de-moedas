
# Conversor de Moedas

![image](https://github.com/user-attachments/assets/e940b7b6-0636-4fbc-9fb0-622a40661e56)


Durante as aulas da imersão alura foi desenvolvido este primeiro projeto que é um conversor de moedas simples que permite aos usuários converter valores de Wons, Dólarares e Euros para Reais.

## Plataforma Utilizada
- CodePen: Plataforma online que oferece um ambiente de desenvolvimento interativo para criar, testar e compartilhar trechos de código front-end (HTML, CSS e JavaScript) diretamente no navegador.
- URL Original: [https://codepen.io/JoyceCruz05/pen/NPWEdab](https://codepen.io/JoyceCruz05/pen/NPWEdab).


## Funcionalidades

- Conversão de Won para Reais
- Conversão de Dólar para Reais
- Conversão de Euro para Reais
- Interface de usuário interativa com prompts
- Exibição do resultado formatado em Reais

## Como usar

1.  Abra o arquivo `https://codepen.io/JoyceCruz05/full/NPWEdab` em seu navegador.
2.  Clique no botão "Bora Converter!".
3.  Siga as instruções nos prompts para inserir a moeda e o valor.
4.  O resultado da conversão será exibido na tela.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Exemplos

- Entrada: 100 Dólares
- Saída: R$ 595,00

- Entrada: 50 Euros
- Saída: R$ 325,00

- Entrada: 1000 Wons
- Saída: R$ 4,00
